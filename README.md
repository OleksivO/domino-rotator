# Domino Rotator
This repository contains source code for Domino Rotator

# Technologies used:
* Angular 1.6
* ES6
* LESS
* Webpack

# Run project:
* Install webpack:
npm install -g webpack
* Install lite-server:
npm install -g lite-server
* Install yarn:
npm install -g yarn

* Run command: yarn
* Run command: npm start
